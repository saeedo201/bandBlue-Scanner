# bandBlue-Scaner

أداة فحص مواقع ويب لاكتشاف ثغرات شائعة (مثل SQL Injection)، واكتشاف ملفات حساسة ولوحات إدارة. الأداة مخصّصة لأغراض التعليم، البحث الأمني، والاختبار المصرّح به فقط.

---

## نبذة
`bandBlue-Scaner` تقوم بمسح شامل لموقع ويب بهدف:
- اكتشاف معاملات قابلة للحقن (SQLi).
- العثور على ملفات حساسة مكشوفة (مثل `crossdomain.xml`, `.env`, ملفات النسخ الاحتياطي).
- اكتشاف لوحات تحكم إدارية ونقاط وصول محتملة.
- إخراج تقارير نصية و JSON مع ملخص المخاطر.

استخدمها في بيئات اختبارية أو فقط بعد الحصول على إذن صريح من مالك الهدف.

## ممارسات استخدام آمنـة
- لا تشغل الفحص ضد أهداف إنتاجية بدون إذن كتابي واضح.  
- استخدم معدلات طلب معتدلة (rate limiting) لتجنب إيقاف الخدمة.  
- عند العثور على ثغرة: اتبع سياسة الإفصاح المسؤول ولا تنشر تفاصيل الاستغلال قبل إصلاحها.  
- اختبر أولاً في بيئات معملية (VM، شبكات داخلية، أو مواقع تدريب مثل testphp.vulnweb.com).
